## 属性

| 属性名        | 说明                           | 类型           | 默认值 |
| ------------- | ------------------------------ | -------------- | ------ |
| right         | 相对于屏幕右边的位置           | Number         | 0      |
| bottom        | 相对于屏幕下边的位置           | Number         | 0      |
| hiddenValue   | 下移距离顶部多少时出现或者消失 | Number         | 0      |
| animationTime | 每次移动的间隔时间             | Number(指毫秒) | 5      |
|               |                                |                |        |
