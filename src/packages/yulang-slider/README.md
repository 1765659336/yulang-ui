## 属性

|         属性名          |         说明         |        类型         | 默认值 |
| :---------------------: | :------------------: | :-----------------: | :----: |
|       sliderValue       |      默认初始值      |       Number        |   0    |
|       sliderStep        |   每次移动最小步数   | Number([1,100]整数) |   1    |
| isOpenDigitalModulation | 是否开启按钮调节滑块 |       Boolean       | false  |
