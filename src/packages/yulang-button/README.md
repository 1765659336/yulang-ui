## 属性

|  属性名  |       说明       |              类型               | 默认值  |
| :------: | :--------------: | :-----------------------------: | :-----: |
| disabled | 控制按钮是否禁用 |             Boolean             |  false  |
|   type   |  设置按钮的主题  | String(default、success、error) | default |
|   size   |  设置按钮的大小  |   String(medium、small、mini)   | medium  |
|          |                  |                                 |         |
